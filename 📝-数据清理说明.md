# 数据清理说明

## 方法1：使用批处理脚本（推荐）

1. 编辑 `清理Mock数据.bat`，修改MySQL连接信息：
```batch
set DB_HOST=localhost
set DB_USER=root
set DB_PASS=你的密码
set DB_NAME=proxyhub
```

2. 双击运行 `清理Mock数据.bat`

## 方法2：手动SQL

如果您有Navicat、phpMyAdmin或MySQL Workbench：

```sql
-- 1. 查看当前所有IP
SELECT id, ip, channelName, remark FROM static_proxies ORDER BY id DESC;

-- 2. 删除Mock数据
DELETE FROM static_proxies 
WHERE remark LIKE '%[MOCK]%' 
   OR channelName = '默认通道';

-- 3. 验证结果（应该只剩3个IP）
SELECT id, ip, channelName, remark FROM static_proxies ORDER BY id DESC;
```

## 方法3：使用前端管理界面

1. 登录管理后台
2. 进入"静态住宅管理"
3. 手动删除每个标记为[MOCK]或默认通道的IP

## 预期结果

清理后应该只剩下3个真实IP：
- `127.134.129.183` (985Proxy)
- `192.187.170.53` (985Proxy)
- `250.130.139.91` (测试购买)

## 注意事项

⚠️ **删除操作不可逆，请确认要删除的数据！**

✅ 建议在清理前先备份数据库：
```bash
mysqldump -u root -p proxyhub > proxyhub_backup.sql
```
