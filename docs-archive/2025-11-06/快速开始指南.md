## 🐛 常见问题

### Q1: 端口被占用
**症状**: 启动失败，提示端口已使用

**解决**:
```powershell
# 查找占用端口的进程
netstat -ano | findstr "3000"
netstat -ano | findstr "8080"

# 关闭进程（替换<PID>为实际进程ID）
taskkill /PID <PID> /F
```

### Q2: flag-icons错误
**症状**: 前端白屏，控制台显示"Failed to resolve import flag-icons"

**解决**:
```powershell
cd frontend
npm install flag-icons
# 重启前端服务
```

### Q3: 数据库连接失败
**症状**: 后端启动失败，提示数据库连接错误
