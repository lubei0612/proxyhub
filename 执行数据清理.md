# 清理Mock数据指南

## 问题
数据库中有17个mock数据（标记为[MOCK]或默认通道），只有3个是通过985Proxy真实购买的IP。

## 清理方法

### 方法1：使用MySQL命令行（推荐）

```bash
# 连接MySQL
mysql -h localhost -u root -p proxyhub

# 查看当前所有IP
SELECT id, ip, channelName, remark FROM static_proxies ORDER BY id DESC;

# 删除mock数据
DELETE FROM static_proxies WHERE remark LIKE '%[MOCK]%' OR channelName = '默认通道';

# 验证清理结果
SELECT id, ip, channelName, remark FROM static_proxies ORDER BY id DESC;
```

### 方法2：使用Web界面

1. 登录管理后台
2. 导航到：静态住宅管理
3. 手动删除每个标记为[MOCK]或默认通道的IP

### 方法3：使用API

```powershell
# 预览要删除的数据
$token = "your_admin_token"
Invoke-WebRequest -Uri "http://localhost:3000/api/v1/admin/clean/preview" `
  -Headers @{"Authorization"="Bearer $token"}

# 执行删除
Invoke-WebRequest -Uri "http://localhost:3000/api/v1/admin/clean/mock-data" `
  -Method DELETE `
  -Headers @{"Authorization"="Bearer $token"}
```

## 预期结果

清理后应该只剩下3个IP：
1. `127.134.129.183` (985Proxy购买)
2. `192.187.170.53` (985Proxy购买)
3. `250.130.139.91` (测试购买)

## 注意事项

- ⚠️ 删除操作不可逆，请确认要删除的数据
- ✅ 删除后需要刷新前端页面查看效果
- ✅ 管理后台统计数据会自动更新
