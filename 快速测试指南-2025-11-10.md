## 🔍 常见问题排查

### 问题1: 仍然显示"The API KEY is invalid"

**解决方案**:
1. 检查`.env`文件配置
2. 确保包含：
   ```bash
   PROXY_985_API_KEY=ne_hj06qomI-bmVfaGowNnFvbUk0YzIzMTc2MTQ1Nzk1Mw==
   PROXY_985_ZONE=6jd4ftbl7kv3
   ```
3. 重启Docker服务：
   ```powershell
   docker compose down
   docker compose up -d
   ```

### 问题2: 页面布局仍然显示不全

**解决方案**:
1. 清除浏览器缓存（Ctrl + F5）
2. 或使用无痕模式测试
3. 确认Docker容器使用的是新构建的镜像

### 问题3: 充值/扣除余额功能不可用

**解决方案**:
1. 检查浏览器控制台是否有错误
2. 确认登录的是管理员账号
3. 查看后端日志：
   ```powershell
   docker logs proxyhub-backend --tail 50
   ```

## 💡 测试技巧

### 测试移动端响应式
1. 打开Chrome开发者工具（F12）
2. 点击设备切换按钮（或按Ctrl+Shift+M）
3. 选择不同设备尺寸：
   - iPhone SE (375px)
   - iPhone 12 Pro (390px)
   - iPad (768px)
   - Desktop (1920px)

### 查看网络请求
1. 打开Chrome开发者工具（F12）
2. 切换到"Network"标签
3. 刷新页面
4. 查看API请求是否成功（200 OK）

### 查看控制台错误
1. 打开Chrome开发者工具（F12）
2. 切换到"Console"标签
3. 查看是否有红色错误信息
