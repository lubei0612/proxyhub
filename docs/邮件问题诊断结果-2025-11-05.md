## 🔍 问题诊断

您没有收到验证码邮件的原因已找到：

### ❌ Outlook邮箱问题
```
Error: 535 5.7.139 Authentication unsuccessful, 
basic authentication is disabled.
```

**原因**: Microsoft Outlook账号禁用了基本身份验证  
**解决**: 需要生成并使用"应用专用密码"

### ❌ Gmail邮箱问题
```
Error: 535-5.7.8 Username and Password not accepted.
```

**原因**: Gmail应用专用密码无效或不正确  
**解决**: 需要重新生成应用专用密码

## 🚀 快速配置步骤（QQ邮箱）

### 1. 开启QQ邮箱SMTP服务

1. 访问 https://mail.qq.com 并登录
2. 点击【设置】→【账户】
3. 找到"POP3/IMAP/SMTP"服务
4. 开启"SMTP服务"
5. 按照提示发送短信验证
6. 获取并保存**16位授权码**

### 2. 更新backend/.env

```env
# 主邮箱（QQ邮箱）
MAIL_HOST=smtp.qq.com
MAIL_PORT=587
MAIL_USER=你的QQ号@qq.com
MAIL_PASSWORD=16位授权码
MAIL_FROM=ProxyHub <你的QQ号@qq.com>

# 备用邮箱（可选：163邮箱）
MAIL_HOST_BACKUP=smtp.163.com
MAIL_PORT_BACKUP=465
MAIL_USER_BACKUP=你的163邮箱@163.com
MAIL_PASSWORD_BACKUP=163授权码
MAIL_FROM_BACKUP=ProxyHub <你的163邮箱@163.com>
```

### 3. 重启服务

```bash
# Windows
taskkill /F /IM node.exe
.\启动ProxyHub-最终版.bat
```

### 4. 测试验证码

访问 http://localhost:8080 并测试注册验证码发送
