## ❌ 常见问题排查

### 问题1：前端显示空白页面
**解决方案：**
1. 打开 Chrome DevTools → Console
2. 查看是否有 Vue 错误
3. 检查 Network 标签，查看是否有 404 错误

### 问题2：管理员无法登录
**可能原因：**
- 后端数据库未正确初始化
- 管理员账号未创建

**解决方案：**
```bash
cd backend
npm run seed
```

### 问题3：Dashboard API 返回 500 错误
**已知问题：** 后端代码已修复，但需要重启后端服务

**解决方案：**
1. 关闭所有窗口
2. 重新运行 `启动ProxyHub.bat`

### 问题4：无法访问管理后台
**检查清单：**
- [ ] 使用的是管理员账号（admin@proxyhub.com）
- [ ] 已成功登录
- [ ] 用户下拉菜单中有"管理后台"选项
