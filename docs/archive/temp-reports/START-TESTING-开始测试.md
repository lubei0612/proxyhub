## ✅ 简要测试流程

### 第一步: 基础功能验证（5分钟）
1. ✅ **登录** - 使用上面的管理员账号
2. ✅ **查看仪表盘** - 确认数据显示正常
3. ✅ **导航菜单** - 点击侧边栏各个菜单项

### 第二步: 核心新功能测试（10分钟）

#### 🔥 重点1: 专线代理（全新功能）
- 菜单: `静态住宅 -> 专线代理`
- 验证: 显示"短视频专线"和"直播专线"

#### 🔥 重点2: 用户删除功能（全新功能）
- 菜单: `管理后台 -> 用户管理`
- 操作: 点击任意用户的"删除"按钮
- 验证: 删除确认对话框弹出

#### 🔥 重点3: 快捷充值金额（已修改）
- 菜单: `我的钱包 -> 账户充值`
- 验证: 快捷金额按钮显示: `2000, 5000, 10000, 50000, 100000`

#### 🔥 重点4: 交易记录标签（已修改）
- 菜单: `管理后台 -> 用户管理 -> 点击任意用户"详情"`
- 验证: 标签显示"**全部交易记录**"（不是"近期交易记录"）

#### 🔥 重点5: 客服链接隐藏（已修改）
- 页面: `账户充值`、`动态住宅选购`、`个人中心`
- 验证: 不显示具体Telegram链接，显示"请联系您的客服"

#### 🔥 重点6: 国家城市筛选（动态加载）
- 菜单: `静态住宅 -> 静态住宅管理`
- 操作: 
  1. 点击"国家"下拉框 - 查看是否有真实国家列表
  2. 选择一个国家
  3. 点击"城市"下拉框 - 查看对应的城市列表

#### ⚡ 重点7: 购买性能优化（重要）
- 菜单: `静态住宅 -> 静态住宅选购`
- 操作: 
  1. 选择国家、地区、时长
  2. 点击"购买"
  3. **计时**: 从点击到完成应该在 **6秒左右**（原来20秒）

#### 💰 重点8: 收入趋势（真实数据）
- 菜单: `仪表盘`（管理员）
- 验证: 收入趋势图显示真实充值数据（不是演示数据）

## 🆘 遇到问题？

### 问题1: 无法访问 http://localhost
**解决方案**:
```bash
# 重启前端
docker compose restart frontend

# 等待10秒再访问
```

### 问题2: 登录失败
**可能原因**: 管理员账户不存在

**解决方案1**: 先注册一个账户
1. 点击"注册"
2. 填写信息注册
3. 注册成功后，我帮您升级为管理员

**解决方案2**: 手动创建管理员
```bash
# 运行这个命令创建管理员账户
docker exec -it proxyhub-backend npm run seed:admin
```

### 问题3: 购买/续费IP失败
**可能原因**: 
- 账户余额不足
- 985Proxy API配置问题

**解决方案**:
```bash
# 先通过管理后台给账户加款
管理后台 -> 用户管理 -> 点击"加款" -> 输入金额
```
