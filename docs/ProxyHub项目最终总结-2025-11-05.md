## 🚀 部署指南

### 开发环境

```bash
# 1. 克隆代码
git clone https://github.com/lubei0612/proxyhub.git
cd proxyhub

# 2. 启动Docker服务
docker-compose up -d

# 3. 安装依赖
cd backend && npm install
cd ../frontend && npm install

# 4. 配置环境变量
# 复制 backend/.env.example 到 backend/.env
# 配置数据库、邮件、Telegram等信息

# 5. 运行数据库迁移
cd backend
npm run migration:run
npm run seed

# 6. 启动服务
# 方式1：使用批处理文件
.\启动ProxyHub-最终版.bat

# 方式2：手动启动
cd backend && npm run start:dev
cd frontend && npm run dev
```

### 生产环境

```bash
# 使用Docker Compose
docker-compose -f docker-compose.prod.yml up -d
```
