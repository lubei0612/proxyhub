## ✅ 部署前检查清单

### 腾讯云准备

- [ ] 已购买云服务器（推荐2核4GB，Ubuntu 20.04/22.04）
- [ ] 已分配公网IP
- [ ] 已设置root密码或SSH密钥
- [ ] 可以SSH连接到服务器
- [ ] 已备案域名（可选，建议）

### 985Proxy配置

- [ ] 已有985Proxy账号
- [ ] 已获取API Key
- [ ] 已获取API Secret
- [ ] 账户有充足余额

### 本地准备

- [ ] 项目代码完整（在 `D:\Users\Desktop\proxyhub`）
- [ ] 已安装SSH客户端
- [ ] 可选：已安装WinSCP（文件传输）

## 🆘 常见问题

### Q1: 宝塔面板 vs 命令行，应该选哪个？

**宝塔面板**:
- ✅ 适合：新手、喜欢图形界面、需要管理多个应用
- ⚠️ 缺点：占用约500MB内存

**命令行**:
- ✅ 适合：熟悉Linux、追求极简、只运行ProxyHub
- ⚠️ 缺点：学习曲线陡峭

**建议**: 新手优先选择宝塔面板

---

### Q2: 必须配置CDN吗？

**不是必须，但强烈建议**

**不配置CDN**:
- 访问速度: 取决于服务器位置和带宽
- 带宽成本: 较高
- 抗攻击能力: 较弱

**配置CDN后**:
- 访问速度: 提升80%+
- 带宽成本: 降低60%+
- 抗攻击能力: 显著提升

---

### Q3: 服务器配置如何选择？

**最低配置**（适合测试）:
- CPU: 1核
- 内存: 2GB
- 硬盘: 20GB
- 带宽: 1Mbps
- 月费用: 约80-100元

**推荐配置**（适合生产）:
- CPU: 2核
- 内存: 4GB
- 硬盘: 50GB
- 带宽: 3Mbps
- 月费用: 约150-300元

**高性能配置**（高并发）:
- CPU: 4核
- 内存: 8GB
- 硬盘: 100GB
- 带宽: 5Mbps
- 月费用: 约400-600元

---

### Q4: 部署需要多长时间？

**时间估算**:

| 部署方式 | 首次部署 | 更新部署 |
|----------|----------|----------|
| 自动化脚本 | 30-60分钟 | 10-15分钟 |
| 宝塔面板 | 1-2小时 | 15-20分钟 |
| 手动部署 | 2-4小时 | 20-30分钟 |

**主要耗时环节**:
- Docker镜像构建: 10-20分钟
- 宝塔面板安装: 5-10分钟
- 文件上传: 取决于网速

---

### Q5: 遇到问题怎么办？

**故障排查步骤**:

1. **查看日志**
   ```bash
   cd /opt/proxyhub
   docker compose logs -f
   ```

2. **检查容器状态**
   ```bash
   docker compose ps
   ```

3. **查看文档**
   - [腾讯云Docker部署指南](./腾讯云Docker部署指南.md) - 故障排查章节
   - [README-部署说明](./README-部署说明.md) - 常见问题章节

4. **查看测试报告**
   - [充值审核功能修复报告](./充值审核功能修复报告-2025-11-06.md)
